How it works

Setup Steps

Step 1 — Initialize project
npm init -y
Step 2 — Install Bootstrap
npm install bootstrap@5.3.3
Step 3 — Link CSS
<link rel="stylesheet"
  href="node_modules/bootstrap/
        dist/css/bootstrap.min.css"
/>
Step 4 — Link JS Bundle
<script src="node_modules/bootstrap/
  dist/js/bootstrap.bundle.min.js">
</script>
